清蒸鱼+糙米饭+青菜

低盐 低脂 高蛋白 高纤维 低GI

这是一份营养均衡的老年人午餐组合，富含优质蛋白质、不饱和脂肪酸和膳食纤维，特别适合有心血管疾病风险或需要控制血糖的老年人。

食材清单

  • 主料：
    • 新鲜鱼（鲈鱼/鲫鱼/鳕鱼）：200克
    • 糙米：80克
    • 青菜（菠菜/油菜/芥蓝）：150克
  • 辅料：
    • 生姜：3片
    • 葱：1根
    • 蒸鱼豉油：10毫升
    • 食用油：5毫升

制作步骤

  1. 准备糙米饭：
    • 糙米提前浸泡30分钟
    • 按1:1.5的比例加水煮饭
    • 电饭煲煮饭模式约40分钟
  2. 清蒸鱼：
    • 鱼洗净后在两面各划几刀
    • 鱼身下垫姜片，鱼肚塞葱段
    • 水开后上锅蒸8-10分钟
    • 出锅后淋上蒸鱼豉油和热油
  3. 炒青菜：
    • 青菜洗净切段
    • 热锅少油，大火快炒1分钟
    • 加少量盐调味
  4. 组合装盘：将糙米饭、清蒸鱼和青菜按比例装盘

小贴士

1. 选择鱼刺较少的海鱼如鳕鱼，更适合老年人食用

2. 糙米可提前浸泡或与白米按1:1比例混合，更易消化

3. 蒸鱼时间根据鱼的大小调整，用筷子能轻松穿透鱼肉即可

4. 青菜可选用深色绿叶菜，营养价值更高
